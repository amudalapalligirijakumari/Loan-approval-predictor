PROJECT NAME:'LOAN APPROVAL PREDICTOR'

## Project Scope

The scope of this project is to develop a Machine Learning model that predicts whether a loan application will be approved or rejected based on applicant financial and personal information.

The project includes:

* Loading and analyzing the loan dataset.
* Preprocessing and encoding categorical data.
* Selecting relevant features for prediction.
* Splitting data into training and testing sets.
* Training a Logistic Regression model.
* Predicting loan approval status.
* Evaluating model performance using accuracy and confusion matrix.
* Visualizing loan approval statistics using graphs.

This project is intended for educational and learning purposes to understand the fundamentals of Machine Learning classification and prediction systems.
